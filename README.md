# Espresso
Team project for the lecture `AI융합캡스톤과 창업`.

## Introduction
* __`Marketing assistance web application`__ using GPT3.5 and Stable Diffusion.
## Feature
* __`Create marketing contents `__ based on `STP` marketing strategy.
